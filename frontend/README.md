# 💻 Edgeify Frontend App

This directory contains the React client application for the **Edgeify** learning platform.

## 🚀 Overview

The frontend is a single-page React app bootstrapped with Vite, styled with Tailwind CSS, and powered by React Router v7. It features a complete learner/instructor interface, dynamic dashboards, structured sidebar navigation, watch history, and a responsive media player.

For detailed instructions, routing structure, and full stack information, see the **[Main Workspace README.md](../README.md)**.

## ⚙️ Quick Start

1. Create a `.env` file (refer to the [frontend configuration settings](../README.md#frontend-configuration-frontendenv)).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Launch the Vite dev server:
   ```bash
   npm run dev
   ```

## 🛠️ Main Dependencies

- **react** & **react-dom**: UI component rendering engine (v19)
- **react-router-dom**: Routing and multi-view page navigation (v7)
- **react-player**: Video media player component
- **axios**: Client HTTP requests to the backend server
- **tailwindcss**: Utility-first CSS styling framework
- **react-spinners**: Clean loading screen spinner components
